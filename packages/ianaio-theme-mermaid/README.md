# Ianaio Theme Mermaid

The mermaid components for Ianaio.

## Installation

Add `ianaio/theme-mermaid` to your package:

```bash
npm i @ianaio/theme-mermaid
# or
yarn add @ianaio/theme-mermaid
```

## Swizzling components

```bash
$ npm swizzle @ianaio/theme-mermaid [component name]
```

All components used by this theme can be found [here](https://github.com/ianaio/ianaio/tree/main/packages/ianaio-theme-mermaid/src/theme)
