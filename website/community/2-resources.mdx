# Awesome Resources

A curated list of interesting Ianaio community projects.

## Community documentation {#community-documentation}

The **[Ianaio.community](https://ianaio.community)** site extends the [official docs](/docs) with more complex examples and full articles that recommend best practices and provide solutions to common problems.

It is also a full Ianaio application just like our main site, so customization and experimentation examples are perfectly suited to be hosted there.

:::info

We might allow direct links to be added in our official documentation that point to the community documentation, on a case-by-case basis, if the community finds such links to articles useful.

We are also contemplating further integration into the official documentation at some point in the future.

:::

## Community plugins {#community-plugins}

### Search {#search}

- [ianaio-plugin-lunr](https://github.com/daldridge/ianaio-plugin-lunr) - Ianaio v2 plugin to create a local search index for use with Lunr.js
- [ianaio-lunr-search](https://github.com/lelouch77/ianaio-lunr-search) - Offline Search for Ianaio v2
- [ianaio-search-local](https://github.com/cmfcmf/ianaio-search-local) - Offline/local search for Ianaio v2
- [@easyops-cn/ianaio-search-local](https://github.com/easyops-cn/ianaio-search-local) - Offline/local search for Ianaio v2 (language of zh supported)
- [ianaio-theme-search-typesense](https://github.com/typesense/ianaio-theme-search-typesense) - Ianaio v2 plugin for [Typesense DocSearch](https://typesense.org/docs/latest/guide/docsearch.html)
- [@markprompt/ianaio-theme-search](https://github.com/motifland/markprompt-js/tree/main/packages/ianaio-theme-search) - Ianaio plugin to add generative AI / LLM-powered docs search, powered by [Markprompt](https://markprompt.com).

### Integrations {#integrations}

- [ianaio2-dotenv](https://github.com/jonnynabors/ianaio2-dotenv) - A Ianaio 2 plugin that supports dotenv and other environment variables
- [posthog-ianaio](https://github.com/PostHog/posthog-ianaio) - Integrate [PostHog](https://posthog.com/) product analytics with Ianaio v2
- [ianaio-plugin-moesif](https://github.com/Moesif/ianaio-plugin-moesif) - Adds [Moesif API Analytics](https://www.moesif.com/) to track user behavior and pinpoint where developers drop off in your activation funnel.
- [ianaio-plugin-yandex-metrica](https://github.com/sgromkov/ianaio-plugin-yandex-metrica) - Adds [Yandex Metrika](https://metrika.yandex.ru/) counter for evaluating site traffic and analyzing user behavior.

### Features {#features}

- [ianaio-theme-github-codeblock](https://github.com/saucelabs/ianaio-theme-github-codeblock). A Ianaio v2 plugin that supports referencing code examples from public GitHub repositories
- [docs-to-pdf](https://github.com/jean-humann/docs-to-pdf) - Generate documentation into PDF format, suitable for Ianaio v1 and v2 (including beta versions)
- [ianaio-plugin-sass](https://github.com/rlamana/ianaio-plugin-sass) - Sass/SCSS stylesheets support
- [ianaio-plugin-remote-content](https://github.com/rdilweb/ianaio-plugin-remote-content) - A Ianaio v2 plugin that allows you to fetch content from remote sources
- [@graphql-markdown/ianaio](https://github.com/graphql-markdown/graphql-markdown) - A Ianaio v2 plugin for generating documentation from GraphQL schemas
- [ianaio-plugin-react-docgen-typescript](https://github.com/atomicpages/ianaio-plugin-react-docgen-typescript) - A Ianaio v2 plugin for loading type annotations from `react-docgen-typescript`
- [ianaio-plugin-react-docgen](https://github.com/atomicpages/ianaio-plugin-react-docgen) - A Ianaio v2 plugin for loading type annotations from `react-docgen`
- [ianaio-plugin-module-alias](https://github.com/atomicpages/ianaio-plugin-module-alias) - A Ianaio v2 plugin for quickly aliasing local modules
- [ianaio-protobuffet](https://github.com/protobuffet/ianaio-protobuffet) - Ianaio toolset for Protobuf contract documentation
- [ianaio-prince-pdf](https://github.com/signcl/ianaio-prince-pdf) - Generate PDF with PrinceXML for better font subsetting and ToC features. Support Ianaio v2 sites
- [reianaio](https://github.com/rohit-gohri/reianaio) - A Ianaio preset for integrating OpenAPI documentation into your docs with [Redoc](https://github.com/redocly/redoc)
- [plugin-image-zoom](https://github.com/flexanalytics/plugin-image-zoom) - An Image Zoom plugin for Ianaio 2
- [ianaio-plugin-typedoc](https://github.com/tgreyuk/typedoc-plugin-markdown/tree/master/packages/ianaio-plugin-typedoc) - A Ianaio 2 plugin to build documentation with [TypeDoc](https://typedoc.org/)
- [ianaio-openapi-docs](https://github.com/PaloAltoNetworks/ianaio-openapi-docs) - A Ianaio v2 plugin and theme for generating interactive OpenAPI docs
- [ianaio-post-generator](https://github.com/moojing/ianaio-post-generator) - A command line tool for user to add a blog/doc file quickly by command like `yarn gen-post new [template] [post_name]`.
