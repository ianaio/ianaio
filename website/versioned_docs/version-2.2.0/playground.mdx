# Playground

Playgrounds allow you to run Ianaio **in your browser, without installing anything**!

They are mostly useful for:

- Testing Ianaio
- Reporting bugs

Use [ianaio.new](https://ianaio.new) as an easy-to-remember shortcut.

Choose one of the available options below.

```mdx-code-block
import {PlaygroundCardsRow} from '@site/src/components/Playground';

<PlaygroundCardsRow />
```

:::tip

For convenience, we'll remember your choice next time you visit [ianaio.new](https://ianaio.new).

:::
